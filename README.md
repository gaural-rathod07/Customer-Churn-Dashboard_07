# Telco Customer Churn – Retention Intelligence Dashboard

## Overview
Interactive dashboard analysing IBM Telco Customer Churn (7,043 customers). 
Coordinated bar chart, histogram, heatmap & map reveal why customers leave. 
Click any city to filter all views instantly.

## Live Dashboard
[View on Tableau Public](https://public.tableau.com/app/profile/gaural.rathod/viz/CS5803_2564542/CustomerChurnOverviewDashboard)

## Dataset
IBM Telco Customer Churn Dataset (v11.1.3+)  
Source: [Kaggle](https://www.kaggle.com/datasets/alfathterry/telco-customer-churn-11-1-3)

## 🛠️ Tools Used
- Tableau Desktop
- Power BI Desktop

## Dashboard Views
| Chart | Question |
|-------|----------|
| Bar Chart | Why are customers leaving? |
| Histogram | How are monthly charges distributed? |
| Heatmap | Which segments have highest churn rates? |
| Map | Where are we losing the most customers? |

## Key Insights
- Competitor issues drive **45% of churn** (841 customers)
- Customers paying **$70+/month** churn significantly more
- **Fiber Optic + Month-to-Month** is highest-risk segment at 58.8%
- **San Diego** has the highest churned customer concentration (185)
